Summary: Latitude vs. X

This dashboard is a data visulaization tool created to support weather changes analysis based on geographical distance from the equator.The analysis revealed that the closer to the equator the more rain and higher temprature we get as well as other accompanying weather conditions. This analytical dashboard is based on data obtained from OpenWeatherMap API to create a dataset on over 500 cities.

Matplotlib was deployed to plot weather data based on latitude, temperature, cloudiness, wind speed, & humidity as an integral parts of the analysis & interpertations.
